# Телеграм-бот для напоминаний о физической активности

## Описание
Чат-бот присылает уведомления со случайно выбранным упражнением каждый час в соответствии с расписанием.

## Команды

### Основные команды
- `/kettlebell` - Получить случайную тренировку по уровню сложности (morning, easy, medium, hard)
- `/weight` - Добавить запись о весе (inline клавиатура)
- `/weight_` - Вывести график веса за период
- `/weight_save` - Получить файл с данными о весе
- `/save_workout` - Получить файл с данными о тренировках
- `/street_wo` - Записать подтягивания и отжимания (inline клавиатура)

### Команды управления тренировками
- `/add_workout` - Добавить новую тренировку
- `/remove_workout` - Удалить тренировку
- `/list_workout` - Просмотреть все тренировки

### Функции

#### Inline клавиатуры
Ввод веса и street workout использует inline клавиатуры с цифрами и кнопками действий.

#### Done/Another кнопки
После получения тренировки доступны две кнопки:
- ✅ **Done** - Отметить тренировку как выполненную
- 🎲 **Another** - Получить другую случайную тренировку
